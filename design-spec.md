# Stuora Bar — QR Menu · Design Spec

Принятые решения по дизайну и реализации. Ориентир брифа: **промо-сторис ≠ меню**.
Стиль бара взят, но *очищен*: один сильный знак, строгая плоская типографика, воздух.
(Ролевые модели: Dead Rabbit, Connaught Bar, Employees Only — люкс через сдержанность.)

---

## 1. Что оставили / починили / выбросили

| Решение | Как реализовано |
|---|---|
| **Оставили** бордо-бархат + золото | Радиальный градиент фона + тонкие золотые ар-деко линии/уголки |
| **Оставили** неоновый рукописный «Stuora» | Единственный ключевой знак — в шапке, розово-неоновое свечение |
| **Оставили** морозный/glass-эффект | Frosted-glass подсветка активной вкладки (`backdrop-filter` + мягкий glow) |
| **Починили** цену | Плоский золотой сериф (Playfair), валюта приглушена (`.cur` — мельче, полупрозрачна) |
| **Починили** тело текста | Кремовый `#F5E6C8`/`#cbb794` на тёмном; золото — только акценты |
| **Починили** декор | Убрано ~70% украшений: осталась рамка + 2 разделителя со «звездой»-искрой |
| **Выбросили** 3D-текст с фаской | Только плоская типографика |
| **Выбросили** золотой медальон + герб | Ни медальона, ни короны/лавра — один неон-скрипт |
| **Выбросили** AI-рендеры напитков | Держимся на типографике, цвете, золоте; фото — опционально позже |

## 4. Компоненты

- **Шапка:** `Since 2024` (трекинг-капитель) · неон-лого · тэглайн · дух места · ар-деко разделитель со «звездой».
- **Sticky-вкладки** (`.tabs`): Cocktails · Infusions · Bites · Non-Alcoholic. Frosted-glass фон, горизонтальный скролл. Тап → плавный скролл к секции. Активная — золотится (рамка + glow), автоцентрируется в баре.
- **Секция:** заголовок (Cinzel) · ступенчатый золотой разделитель (ромб-шеврон) · цена категории «Each 11€» (золотой сериф).
- **Карточка** (`.card`) — интерактивная, «тап → раскрытие»:
  - **Свёрнуто:** нарисованный бокал (SVG-линия) · название · опц. «✦ Signature» · опц. цена справа (только если у позиции своя, иначе — из шапки секции) · золочёный шеврон.
  - **Развёрнуто (аккордеон):** описание + список ингредиентов (`item.ingredients`, когда заполнен из меню-документа; иначе показывается описание). Раскрытие — плавное (grid-rows `0fr→1fr`), одновременно открыта только одна карточка.
- **Система бокалов** (вместо фото): 7 типов линией-арт — `shot · rocks · highball · martini · coupe · mug · plate`. Тип и «цвет вкуса» задаются в `menu.json` (`glass`, `tint`). Настойки → шот, коктейли → бокал по подаче, закуски → «тарелка» (без жидкости). Данные — `app.js` (объект `GLASS`).
- **Подвал:** имя (скрипт) · адрес (вход с Rīdzenes) · телефон (кликабельный `tel:`) · часы (Чт/Вс 18–01 · Пт/Сб 18–02 · Пн–Ср приватно) · «Walk-ins welcome». Верхняя золотая линия — плашка-латунь.

**Флагманы-сигнатуры:** механика data-driven — поле `"signature": true` в `menu.json`.
Сейчас помечены 3: *Tomato & Sprats Mary*, *Mushroom Old Fashioned*, *Smoked Plum Vodka*.

## 5. Анимации (деликатно, без библиотек)

- **Neon breathe:** лёгкая пульсация свечения лого, 5.5 s.
- **Бокал «рисует себя»:** контур SVG прочерчивается золотом при появлении (`stroke-dashoffset`, `pathLength="1"`), 1.1 s — фирменный ар-деко-момент.
- **Налив «цвета вкуса»:** жидкость поднимается снизу однократно (`transform: scaleY`, `transform-box: fill-box`), 0.85 s. Без плескания/петель.
- **Тап → раскрытие:** аккордеон `grid-template-rows 0fr→1fr` + fade, 0.38 s, одна карточка за раз.
- **Reveal-on-scroll:** `IntersectionObserver` → карточки появляются fade + translateY(18px→0), 0.7 s, единожды.
- **Scroll-spy:** второй `IntersectionObserver` золотит активную вкладку по видимой секции.
- **Frosted glass:** `backdrop-filter: blur` на sticky-вкладках + мягкий glow на активной.
- **Smooth scroll** по тапу вкладки.
- Уважается `prefers-reduced-motion`: пульсация и сдвиг карточек отключаются.

## 6. Технически

- Ванильный HTML + CSS + JS, ноль зависимостей (кроме Google Fonts).
- Данные — `fetch("menu.json")`; позиции **не** захардкожены. Ошибка загрузки → аккуратный fallback.
- Мобайл-фёрст, колонка `max-width: 480px`, `100dvh`, `env(safe-area-inset)`.
- Слот фото заложен (`item.image` → `photos/<файл>`), при ошибке загрузки — просто убирается.
- **Проверено локально:** `python -m http.server` → index/CSS/JS/JSON отдаются `200`, меню рендерится из `menu.json` (headless-скриншоты сняты).
